# CORSair

A browser-based **CORS misconfiguration** testing tool.

<img width="1365" height="634" alt="Screenshot" src="https://github.com/user-attachments/assets/1139c72a-ddc7-4052-8dc7-1ae7fa833873" />


## About
CORSair checks whether a target server has a permissive cross-origin resource sharing (CORS) policy by attempting an authenticated cross-origin request from your browser and reporting the result.

## Usage
1. Open the tool: https://sooryaggokul.github.io/CORSair/
2. Enter the target URL.
3. Toggle "Include Cookies" if you want to test with credentials.
4. Click **Fetch Response** to see the result.
